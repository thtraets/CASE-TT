Note voor gebruik
- De meeste browsers zullen de hostende api eerst niet vertrouwen. Oplossing: CORS is geregeld in de applicatie, maar in de browser moet nu eerst even de api host worden bezocht om aan te geven dat de site te vertrouwen is. Dit hoeft alleen de eerste keer, daarna kan de frontend aan de slag. 

1. Ga naar https://localhost:5001. De browser zal een waarschuwing geven, kies ervoor om de api endpoints toch te vertrouwen. 
2. Ready to go :)
